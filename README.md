# cityestates

Playground. Real estate application.
Technologies used:
 - Django framework
 - PostgreSQL
 - Bootstrap 4
 - Python
 - CSS
 - HTML
 - Windows 10

 Deployed with Gunicorn and Nginx.

 ## Run server
Activate virtual environment.

```bash
.\venv\Scripts\activate
```

Run server

```bash
python manage.py runserver
```
